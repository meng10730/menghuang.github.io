# 特殊能力與戰鬥機制設定

## 壹、力量體系之基礎邏輯

1. **力量源泉：靈力**
   天地間的靈力為一切非常之力的根源。

2. **普及度與流派分支：大眾皆可修習**
   此等力量並非少數天選之子獨有，而是世人皆可學習的系統。正因其普及，歷代傳承之下演化出繁多不同的「流派」與「分支」。

3. **提升之道：殊途同歸**
   力量之提升並非單一路徑，而是擁有多種方式（如肉體打磨、靈力累積、心境頓悟等）。這些不同的提升途徑，亦造就了各分支在修練上的獨特風格。

4. **百變一宗：核心機制與變體**
   - **常人界限：** 一般修行者大多僅能擁有一脈專屬的「核心機制」。然而，透過對自身機制的深入理解，可由單一核心衍伸出千變萬化的「變體」應用。
   - **破格存在：** 少數身懷大能者或異方霸主（如強大的神獸、敵對首領），方能跳脫常理，同時持有多種截然不同的機制。

5. **萬法歸一：共同鐵則**
   天地有常，世間存在著所有靈能使用者皆無可違逆的「共同鐵則」（註：為了涵蓋所有情況，鐵則體系將包含多項條規，目前確立基礎兩項，其餘待增補）：
   - **靈能守恆定律（業報與代價）：** 任何機制的發動皆須消耗等量靈力或特定資源。若強自發動超負荷的變體或因果秘術，差額將由施術者的「生機、壽命或修為」中強制扣除，引發乾坤反噬。
   - **同態相斥規矩（領域衝突）：** 兩種「相似」或「同源」的核心機制，無法在同一現世之中完美並存。若雙方使用相似能力，會產生靈力干涉導致互相削弱，逼迫戰鬥走向「變體」的變招博弈。

6. **高下立判：階級劃分**
   修練之境以「能力覆蓋的空間與陣法擴張程度」為核心，分為五大階級：
   - **【執寸】（方寸之間）：** 能力發動與影響範圍僅限於自身的皮膚與手持武器的接觸面。
   - **【羅身】（包羅周身）：** 能將機制擴展到周身兩三公尺處，形成最基礎的應對領域與氣場。
   - **【布陣】（掌控戰局）：** 戰鬥的真正分水嶺。能將自身機制的規則覆蓋整個戰場，把對手強制拉入自己制定的「機制遊戲」中。
   - **【覆天】（天崩地裂）：** 里程碑階級，擁有該階專屬的「通用反制手段」。能改寫現世常規與靈力流動。
   - **【無間】（無遠弗屆）：** 力量不受空間所限，甚至能跨越極大距離或干涉時間發動，為近乎神明的境界。

## 貳、機制的複雜度與規則設計

1. **複雜度與可讀性平衡**
   機制的設定不會過度繁瑣以致影響閱讀體驗，而是保持一定難度的邏輯推演空間。戰鬥的魅力在於規則的靈活運用、情報戰與心理博弈，而非單純的數學公式推導。

2. **矛盾判定本源**
   當兩種相互衝突的機制（如必中之矛對上無二防護之盾）碰撞時，勝負判定主要考量兩大基準：
   - **屬性生剋**：萬物皆有其天敵，屬性上的克制具有極高優先權。
   - **原初與衍生（發明時間先後）**：機制的「古老程度」。越是本源、古老的規矩（初代發明者），在概念對撞時通常優於後世的衍生或臨摹機制；反之，若新興機制是專門為「破解」古老規矩而生，則具備特攻效果。

3. **制約與誓約（代價）**
   此界存在「制約」概念。若願意揹負慘痛的代價或立下嚴苛不可違逆的誓言，便能換取打破同階級常規、霸道無解的機制能力。

4. **本相神通/因果秘術**
    世間確實存在足以改易現世常規或干涉因果的「本相神通/因果秘術」大能。但為維持天地平衡，這類能力無法輕易施展，必須掛上「具體、嚴苛的特定條件或前置儀式」做為限制方可發動。

5. **反制與通用封印手段**
    - **初階通用解**：對於較為基礎的術法與機制，存在共通的破解之道（例如打斷靈力流動、破壞術式節點）。
    - **階級里程碑的「集大成之技」**：當修行者跨入特定的高階「階級」時，會領悟或掌握該位階特有的「通用反制或封印手段」。
    - **平衡限制**：為避免通用解過於氾濫導致機制戰失去意義，此類「階級通用封印手段」的數量會受到嚴格限制（例如全作僅有三大封印法或五大破法領域）。

## 參、情報戰與推演邏輯

1. **主角戰鬥風格**
    主角戰鬥風格偏向「絕境中尋找機制漏洞的臨場反殺型」。不依賴力量碾壓，而是在劣勢與高壓的戰況中，透過感官、觀察力與臨場反應，找出敵方機制中看似無敵的致命盲點，達成逆轉。

2. **情報的價值**
    在「機制對決」中，情報的價值極高。一旦自身的機制底細與發動條件被看破，形同「必敗」，因此需要盡量隱藏底牌；反之，看破對手後，仍須經歷一番推演與佈局（例如引導對方進入特定現世環境）才能將優勢轉化為勝局。

3. **試探與騙局的博弈**
    雖然中低階修行者的戰鬥較為直接，但在「高手/高階層」的對決中，充滿了心理博弈與騙局。刻意釋放假情報、偽造虛假的發動條件（例如明明靠聲音發動，卻每次都假裝結印）以誤導對手對機制的推演，是強者間常見的高端戰術。

4. **解說與演出形式**
    為提升讀者代入感與燒腦感，機制解說將主要由「戰鬥雙方在心裡反覆推算、倒推對手能力」的方式呈現（如：A在心裡猜測B，B又猜到A在猜測自己）。讓讀者跟隨著角色的視角，一邊打鬥一邊經歷從困惑、試探到豁然開朗的解謎快感。

5. **環境互動機制**
    戰鬥並非發生在真空環境中。戰場的周遭元素（地勢、天候、甚至是旁觀的凡人等）都能與角色的機制產生強烈互動，為戰鬥增添不可控的變數，甚至成為雙方爭奪的破局關鍵。

## 肆、代價與副作用

1. **常規與特殊副作用**
    一般修行者正常發動符合自身階級的機制，僅有靈力與體力的常規消耗。然而，少數具有「特殊設定」的人物或異常的機制（例如偏枯之氣），在過度使用時會產生專屬的生理或心理副作用（例如使用冰系將逐漸失去人類情感）。

2. **代價侵蝕與時間壓力**
    當修行者發動超負荷的強力機制或揹負「制約」時，所付出的代價並非戰後統一結算，而是「戰鬥中隨著時間流逝不斷侵蝕身心」。這種設定強制讓強大絕招附加了「光陰如沙流逝之催折」。

3. **終極威懾（禁忌機制）**
   世間確實存在「一經發動便能毀天滅地，但使用者也將迎來形神俱滅」這類保證同歸於盡的同歸之術。雖數量極少，但在劇情中常被少數頂尖強者或偏執勢力做為「最終威懾」，擁有改變世界格局的戰略意義。
